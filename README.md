☕ Cafe Order Dashboard

A modern and responsive **Cafe Order Dashboard** built using **React, TypeScript, Vite, and Tailwind CSS**. The application helps manage cafe orders with a clean UI, fast performance, and an intuitive user experience.

⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/suryanshkumar07/cafe-order-dashboard.git
```

Go to the project folder:

```bash
cd cafe-order-dashboard
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```
